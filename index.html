not a knowledge class

這不是一堂
「學東西」的課。

這不是知識課。也不是療癒課。

這是一段你親自走過的過程。透過身體、覺察與練習，重新看見自己。不是聽懂就算了，而是真的在每一站裡，跟自己碰面。

不是得到更多答案，
而是開始看見：
自己本來就在那裡。

same search

每個人其實都在找同一件事。

從身體找

有人先感覺到的是累、緊、痛、失衡。於是他從身體開始往回找。

從心找

有人是先感覺到空、亂、沒有方向。於是他從內在開始靠近自己。

從世界找

有人先在關係、環境與世界裡卡住。於是他開始找自己的位置。

但最後會發現：
答案都會回到自己。

不用努力追尋，
只是走一圈再回到自己。

點每一站，看看這段旅程怎麼走。這份課程不是線性升級，而是一個循環。

journey map心

這是一條讓時間變成路徑，
讓路徑成為看清自己源貌的旅程。

stage 1

心｜開始看見自己

課程從心開始。不是要你立刻解決什麼，而是先慢下來，看見自己現在在哪裡。

這一站的重點不是答案，而是靠近。
first half

前半段，是開始看見自己。

心

開始看見自己的內在。不是急著定義自己，而是開始靠近自己。

實作

開始看見身體、行動與反應。很多你以為只是習慣的事，其實都在說話。

自我

開始理解自己是怎麼形成的，平常又是怎麼運作的。

world

當人開始看見自己，
也會開始看見世界。

這裡說的世界，不是抽象的大道理。而是土地、環境、人、關係。

這一堂不是理解世界，而是看見：自己在世界裡的位置。

土地

你站在哪裡，會影響你怎麼活。

環境

你如何被空間、節奏與氣氛影響。

關係

你如何在他人之中失去自己，或找回自己。

世界打開之後，
人會重新回頭看自己。

inner observation

第二次看見自己。

但這一次的觀察，不再只是看身體，而是開始內觀。

透過冥想與覺察，看見內在的運作。很多人會第一次發現：自己其實一直在跟自己對話。

move through the world

理解自己之後，
還要學會怎麼在世界裡行走。

攻擊與回應

不是所有力量都要硬接。先分辨，才知道怎麼回。

借力使力

真正的力量不一定是對撞，而是知道怎麼用力。

穿越阻礙

不是打贏世界，而是用自己的節奏，穿過它。

return to heart

走完一圈之後，
會回到最初的地方。

心。

但這一次再看自己的心，通常已經完全不同。學員會再做一次第一堂做過的事：再畫一次自己的心。

很多人會在這時候第一次發現：自己其實已經走了很遠。

這不會讓你變成更好的人
而是和自己靠在一起

body collaboration

當心重新被看見，
身體也開始被邀請回來。

點一下不同部位，看看這段課程會怎麼邀請身體開始合作。

body practice

頭腦｜你最近在忙什麼？

這堂課不是要你把頭腦關掉，而是讓它不再一個人扛所有事。當頭腦不用一直控制，身體才有機會說話。

「你可以先不用解釋。先感覺就好。」
last class

最後一堂課，
會停用精油。

我們會走到戶外，回到土地。摸摸地球。不是離開課程，而是把整段旅程真正帶進生活。

天啊，地啊，我在這裡。
我來的時候沒有通知你。
但從現在開始，我會好好活在這裡。
我會聽你，也會聽我。
跟你配合。
請多多指教。
dual mentors

讓身體跟得上靈魂的變化

這堂課由兩位講師共同帶領。
我們相信，身、心、靈不是分開提升的三件事，而是需要一起被照顧的三個容器。

很多時候，心已經想往前了，靈魂也已經開始變化，但身體還停留在原地。

因此，身為芳療師的我們，選擇以精油作為工具，讓課程成為路徑，也讓身體不再只是跟不上，而能真正參與這段轉變。

精油不只是輔助，也是我們一起工作的夥伴。你甚至可以把它理解成：當你回到家之後，仍然陪你繼續練習的另一位講師。

植物的自由性，有時比語言更能支撐一個人，走過那些正在發生、也正在改變的過程。
